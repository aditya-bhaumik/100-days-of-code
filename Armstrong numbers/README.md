<h2><a href ="https://www.geeksforgeeks.org/problems/armstrong-numbers2727/1">Armstrong Numbers</h2>
For a given 3 digit number, find whether it is armstrong number or not. An Armstrong number of three digits is an integer such that the sum of the cubes of its digits is equal to the number itself. Return "Yes" if it is a armstrong number else return "No".
<h2>Example 1 : </h2>
<pre>
  <b>Input :</b> N = 153
  <b>Output :</b> "Yes"
  <b>Explanation :</b> 153 is an Armstrong number
since 1<sup>3</sup> + 5<sup>3</sup> + 3<sup>3</sup> = 153.
Hence answer is "Yes".
</pre>
<h2>Example 2 :</h2>
<pre>
  <b>Input :</b> N = 370
  <b>Output :</b> "Yes"
  <b>Explanation :</b> 370 is an Armstrong number
since 3<sup>3</sup> + 7<sup>3</sup> + 0<sup>3</sup> = 153.
  Hence answer is "Yes".
</pre>
<b>Your Task: </b> 
  
You dont need to read input or print anything. Complete the function armstrongNumber() which takes n as input parameter and returns "Yes" if it is a armstrong number else returns "No"..
